# Haskell Academy

The open-source web app that is used to run Haskell Academy.

## Usage

Front-end Development Server
```bash
npm start
```

API Development Server
```bash
npm run api
```

Other tasks:
```bash
# Just build the frontend app into `dist/`
npm run build
# linting
npm run lint
# unit tests
npm test
# clean dist directory
npm run clean
```
